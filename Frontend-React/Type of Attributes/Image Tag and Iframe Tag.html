Iframe is used to upload the google map address Link and youtube link

Steps to upload the Google Maps :
1. Select and search the address in the google maps and then Click the share button
2. Click on Embed a Map option and copy the html address
3. Paste the html address in the VS Code using Iframe Tag
4. Therefore the addres will be shown and width and height can be ajdusted in the html

Steps to upload the Youtube(steaming) Video Link :
1. Select the youtube video you wnat to upload in the html
2. Play the video and let the AD complete & Click on share button
3. Then select the Embed Link and copy the html link
4. Paste the html address in the VS Code using Iframe Tag
5. Therefore the addres will be shown and width and height can be ajdusted in the html
